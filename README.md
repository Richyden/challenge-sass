# SASS Challenge

This was a solo project carried out on the 20th of January 2021 as part of the BeCode Li√®ge's Junior Web Developer programm. We were tasked to remake an css code with SCSS.

## Learning objectives
 * Being able to explain what a CSS-Preprocessor is
 * Generate some CSS from your CSS preprocessor (SASS)
 * Being able to minify your CSS output
 * Knowledge of the following SASS features :
 	* Variables
 	* Mixins (Functions)
 	* Nesting
 	* Partials & Import
 	* Extend / Inheritance
 	* Operators (Math)
